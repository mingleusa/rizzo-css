# Rizzo CSS Design System

This Storybook showcases the **Rizzo CSS** design system. All components use the same **BEM** class names and **semantic CSS variables** that ship with the `rizzo-css` package.

## Same design system, every framework

The styles and components you see here are **React** implementations. The same look and behavior are available for:

- **Astro** — reference components and docs at [rizzo-css.vercel.app/docs](https://rizzo-css.vercel.app/docs)
- **Svelte** — `/docs/svelte`
- **React** — this Storybook and `/docs/react`
- **Vue** — `/docs/vue`
- **Vanilla** — HTML + same BEM classes + `js/main.js` — `/docs/vanilla`

One CSS bundle, one design system. Use the framework that fits your project.

## What’s in this Storybook

- **Introduction** — this page
- **Rizzo/*** — individual component stories (Button, Badge, Card, Alert, Modal, Tabs, etc.)
- **All components** — browse all React components (50+) from one dropdown
- **Blocks** — pre-built layouts (Landing hero, Pricing, Dashboard, Docs)

## Getting started

- Install: `pnpm add rizzo-css` (or npm/yarn)
- Import CSS: `import 'rizzo-css'` or `<link href="…/rizzo.min.css" rel="stylesheet">`
- Use the same class names in your templates; copy examples from the docs or from these stories.

Full docs: [rizzo-css.vercel.app](https://rizzo-css.vercel.app)
