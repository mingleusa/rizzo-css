Why use Rizzo CSS

  • Framework-agnostic

    One design system for Vanilla, Astro, Svelte, React, and Vue. Same BEM classes and tokens everywhere.

  • Accessible by default

    Components follow WCAG guidance: focus, ARIA, keyboard nav, and reduced motion built in.

  • Themable

    Light, dark, and custom themes via CSS variables. No runtime lock-in.
